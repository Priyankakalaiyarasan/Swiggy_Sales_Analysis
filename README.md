# 🍔 Swiggy Sales & Delivery Performance Dashboard

## 📌 Project Overview
This project focuses on analyzing **Swiggy sales and delivery performance** using a dataset of **5000 rows**.  
The goal is to derive insights into sales trends, delivery efficiency, and customer satisfaction using **Microsoft Power BI**.

---

## ❗ Problem Statement
Food delivery platforms generate massive amounts of data daily.  
However, extracting meaningful insights from raw data is challenging.

**Key challenges:**
- Understanding city-wise sales performance  
- Identifying delivery delays  
- Analyzing customer ratings  
- Improving operational efficiency  

---

## 🎯 Project Objectives
- Analyze total sales by city and restaurant  
- Evaluate delivery time performance  
- Identify top-performing cities and items  
- Measure customer satisfaction using ratings  
- Build an interactive dashboard  

---

## 📊 Dataset Information
- Total Rows: **5000**
- Columns:
  - Order_ID
  - Order_Date
  - City
  - Restaurant_Name
  - Category (Veg/Non-Veg)
  - Item_Name
  - Quantity
  - Price
  - Delivery_Time (mins)
  - Rating
